Front end Development

Web Design
Case

Voor het vak Front end Development hebben we de opdracht gekregen om twee webpagina's na te maken voor op de mobiel. Voor deze opdracht heb ik gekozen om het Louvre museum na te maken, omdat ik van de zomer de kans heb gehad om de welbekende schilderij van Leonardo da Vinci te zien de Mona Lisa. Aan de hand van HTML, CSS en JavaScript heb ik de website zo identiek mogelijk gebouwd.   

Louvre

De website heeft een elegante layout waar ik goed mee kan werken en heeft tegelijkertijd ook zijn eigen moeilijkheidsgraad.

De home-page en de Mona Lisa pagina zijn de twee onderdelen die ik heb gekozen om na te bouw. Ik heb gekozen om mijn focussen op de surface neer te leggen. 

Gloria Kwako
